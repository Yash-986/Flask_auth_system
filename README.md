# Flask Authentication System

A simple and secure authentication system built with Flask.

# Features
- User registration
- Login / Logout
- Password hashing
- Protected routes
- Flask-Login integration

# Setup

```bash
git clone https://github.com/Yash-986/flask-auth-app
cd flask-auth-app
pip install -r requirements.txt
export SECRET_KEY="your-secret-key"
python run.py
